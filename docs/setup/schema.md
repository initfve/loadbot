
# Defining schemas


### Schema fields

- `name` - unique name, used in jobs (see job.schema) for determining which template use
- `database` - database name
- `collection` - collection name
- `schema` - actual document template

### Schema document template fields

General
- `#id` 
- `#string`
- `#word`

Internet
- `#email`
- `#username`
- `#password`
 
Person
- `#name`
- `#first_name`
- `#first_name_male`
- `#first_name_female`
- `#last_name`
- `#title_male`
- `#title_female`
- `#phone_number`
